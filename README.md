# Sepium Test Task

Тестовое задание Sepium Group (От Савельева Андрея)

## Демо

Тест проекта доступен по ссылке: https://sepium-test-task.netlify.app/

## Стек

- HTML
- CSS
- JavaScript
- jQuery
- Swiper
- Fancybox

## Что реализовано

- листинг карточек товаров
- адаптивная верстка для десктопа и мобильных устройств
- цикличный слайдер изображений
- модальное окно просмотра изображений
- переключение модификаций
- переключение состояния like
- базовая SEO-разметка
- использование `sprite.svg`
- частичный перевод размеров на `rem`

## Локальный запуск

Открыть [`index.html`](./index.html) в браузере.

Для корректной работы удобнее запускать проект через локальный сервер, например:

```bash
npx serve .
```
или live-server
